Sudoku Game Readme

This Sudoku game is implemented in Python using the Pygame library for the graphical user interface. Below are instructions on how to compile and execute the provided code.

Prerequisites:
- Latest Python installed on your system
- Pygame library installed (you can install it via pip: `pip install pygame`)

Instructions:

1. Download the provided Python script named "sudoku.py" to your local machine.

2. Ensure that you have Python installed on your system. You can download Python from the official website: https://www.python.org/downloads/

3. Install the Pygame library by running the following command in your terminal or command prompt:
   
 pip install pygame

4. Once Pygame is installed, navigate to the directory where you saved the "sudoku.py" file using the terminal or command prompt.

5. To run the Sudoku game, simply execute the Python script by typing the following command and pressing Enter:

 python sudoku.py

 
6. The Sudoku game window should now open, allowing you to interact with the game using your mouse and keyboard.

7. You can start a new game, select difficulty levels (easy, medium, or hard), input numbers into the Sudoku grid, solve the puzzle, undo moves, and quit the game using the buttons provided in the GUI.

8. Follow the on-screen instructions and enjoy playing Sudoku!

Note: Make sure to resize the game window if needed to fit your screen properly.
